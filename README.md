# Eclipse BDR Plugin

Claude skills and agents for the Eclipse BDR team. Built and maintained by AI Institute.

## What's inside

### Skills

- **regulatory-scan-by-source** — scans regulatory sources for BDR-relevant signals.

### Agents

## Installation

In Claude Code or Cowork:

```
/plugin marketplace add instituteai-io/eclipse-bdr-plugin
/plugin install eclipse-bdr-plugin@eclipse-bdr-plugin
```

## Updates

When a new version is released, run `/plugin update` (Claude Code) or use the plugin manager UI (Cowork) to pull the latest.

## Versioning

This plugin follows semantic versioning. The current version is in `.claude-plugin/plugin.json` and `.claude-plugin/marketplace.json` — keep them in sync when bumping.
